# Security policy

Please report security issues by emailing security@authkit.dev. We will respond within 72 hours and aim to ship a fix within 14 days for high-severity issues.

We do not accept security disclosures via public GitHub issues.

## Supported versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | yes       |
| < 0.1   | no        |
